MiniHapz To Do List
<p align="center">A simple, clean, and interactive To Do List app built using <b>HTML, CSS, and JavaScript</b>.</p> <p align="center"> <img src="https://img.shields.io/badge/Status-Active-brightgreen?style=flat-square"> <img src="https://img.shields.io/badge/Tech-HTML%20%7C%20CSS%20%7C%20JS-blue?style=flat-square"> <img src="https://img.shields.io/badge/Storage-localStorage-orange?style=flat-square"> </p>

Fitur Utama
- Manajemen Kegiatan
Tambah, lihat, dan hapus kegiatan.
- Tambah Kegiatan
Tambah kegiatan pakai modal yang clean.
- Sub-Kegiatan Dinamis
Bisa nambah banyak sub-kegiatan.
- Detail Kegiatan
Lihat semua sub-kegiatan lewat tombol Detail.
- Hapus Kegiatan
Ada alert konfirmasi.
- Filter 
Filter berdasarkan:
Nama kegiatan
Tanggal kegiatan
- localStorage Support
Semua data disimpan di browser.
Auto load saat dibuka ulang.
Data aman walau refresh.

- Background Music

tombol disebelah To Do List.

Cara Pakai

1. Buka index.html di browser.
2. Klik + Tambah Kegiatan untuk membuat kegiatan baru.
3. Isi nama, list, dan tanggal.
4. Tekan Detail untuk melihat sub-kegiatan.
5. Tekan Hapus untuk menghapus kegiatan.

Enjoy & Have Fun
